# YiiTK

## Models

Para configurar sua classe `Model` estenda `yiitk\base\Model` para habilitar as funcionalidades de **YiiTK** em seus modelos `Model`:

```php

    /**
     * Class Model
     */
    class Model extends \yiitk\base\Models
    {
        
    }

```
